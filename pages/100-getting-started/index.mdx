# Начать игру

![Спавн Плазмо, третий сезон](/assets/getting-started/s3_spawn.jpg) 

На Плазмо можно играть с любого пиратского или лицензионного лаунчера. Версию игры можно узнать [тут](/stats)

Сервер работает на Джава версии Майкрафт (ПК), но поддерживает [кросплатформенную игру с Бэдрок](/guides/bedrock) (Телефоны и консоли)

## Адрес сервера

```
prp.plo.su
```
